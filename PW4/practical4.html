Paragraphs

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.

Task 3

  1. What is selector's specificity and who it's calculated?

    Specificity is a value that determines which CSS rule takes precedence when multiple rules target the same element. It is calculated based on the type of selector used in the CSS rule. Specificity is often represented as a numerical value where higher values indicate higher specificity.

  2. What are complex selectors?

    Complex selectors are selectors in CSS that involve more than one simple selector. They can target elements based on their hierarchy, relationships, attributes, and more. Examples of complex selectors include descendant selectors, child selectors, attribute selectors, and more.

  3. What is the difference between pseudo-classes and pseudo-elements?

    Pseudo-classes are keywords that can be added to selectors to style certain states of elements, such as :hover, :active, :focus, etc. Pseudo-elements, on the other hand, allow you to style parts of an element, like the first line of a paragraph or the first letter of a word, using keywords like ::before, ::after, etc.
