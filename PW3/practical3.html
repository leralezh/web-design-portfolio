  1. Research for and describe the ways to open developers console in Mozilla Firefox and Safari.

      To open the developer console in Mozilla Firefox, you can:
    • Use the keyboard shortcut Ctrl + Shift + J (or Cmd + Shift + J on a Mac).
    • Select “Browser Console” from the Browser Tools submenu in the Firefox Menu (or Tools menu if you display the menu bar or are on macOS).

  2. What way of including internal links into the HTML page you would consider generally preferable: absolute or relative links? Describe use cases for choosing absolute links and choosing relative links.

    • Regarding internal links in HTML, relative links are generally preferable because they are shorter and make the document more portable. They are useful when moving the site to a different domain or when using the same code across multiple environments
    • Absolute links are better when you want to ensure the link will work regardless of the file structure, such as when linking to resources on a different domain or server.

  3. Research and write about image formats used in web-development. What is the difference between JPG, GIF and PNG formats? What is WEBP image type? What is SVG?

      In web development, the common image formats are:
    • JPG: Good for photographs and images with gradients. It uses lossy compression, which reduces file size but can affect quality.
    • GIF: Supports animation and is suitable for simple graphics with solid colors. It uses lossless compression but is limited to 256 colors.
    • PNG: Ideal for images that require transparency or have text and objects with sharp edges. It uses lossless compression and supports more colors than GIF.
    • WEBP: A modern format that provides superior lossless and lossy compression for images on the web.
    • SVG:A vector image format used for icons, logos, and illustrations. It remains crisp at any resolution or size and supports animation and interactivity.
