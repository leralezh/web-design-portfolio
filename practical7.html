    1. Repaint and Reflow:
  • Repaint: Updates visual appearance without changing layout.
  • Reflow: Recalculates layout due to DOM changes.
  • Both impact performance; minimize unnecessary changes
    2. JavaScript and Performance:
  • Download time: Large JS files slow loading.
  • Parsing and execution time: Optimize code.
  • Blocking main thread: Use async techniques
    3. Minification:
  • Removes unnecessary characters from JS code.
  • Shortens variable names.
  • Benefits: Faster loading, saves bandwidth.
    4. Web Workers:
  • Run JS in background threads.
  • Perform tasks without blocking UI.
  • Useful for heavy computations or async tasks.
